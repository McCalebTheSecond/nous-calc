# Nous Calc

Mobile-first **ClevCalc-style** calculator (scientific + unit + currency converters) themed with **Hermes Nous dark** tokens from [`hermes-android`](https://github.com/Techtree-Partners/hermes-android) (`HermesDarkTokens`).

## Features

- Scientific calculator (trig, log, powers, history)
- Unit converter (length, weight, temperature)
- Currency converter (static demo rates)

## Develop

```bash
npm install
npm run dev
npm test
npm run build
```

## Theme

Dark palette: deep Nous blue background (`#0D2F86`), cream foreground (`#FFE6CB`), elevated surfaces (`#12378F`) — matches Hermes Android built-in **Nous** theme.
